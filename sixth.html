What is HTML ?

HTML (HyperText Markup Language) ek markup language hai jo web pages banane ke liye use hoti hai. Yeh kisi programming language jaise C ya Java nahi hai, balki ek structure define karne wali language hai. Aap jo bhi website dekhte ho (Google, Facebook, Amazon), unka structure HTML se banta hai.

  1. Common HTML Tags

    • < h1 > to < h6 >→ Headings (h1 = biggest, h6 = smallest)
    • < p > → Paragraph text.
    • < a href="link" > → Links (anchor tag).
    • < img src="image.jpg" alt="desc" > → Image.
    • < ul >, < ol >, < li > → Lists (unordered / ordered).
    • < table >, < tr >, < td >, < th > → Tables.
    • < form >, < input >, < button > → Forms and inputs.
    • < div > → Container for grouping elements
    • < span > → Inline text container.

  2. Attributes

    < a href="https://www.google.com" target="_blank">Go to Google
    • href = link ka address
    • target="_blank" = link naya tab me open hoga

  3. Why HTML is important ?

    • Har webpage ka skeleton/structure HTML hi hota hai.
    • CSS (design) aur JavaScript (functionality) ke saath milkar complete website banti hai.
